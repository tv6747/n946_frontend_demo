# Frontend Demo Project

This is a frontend demonstration project built with React and Vite.

## Project Structure

- **Framework**: React + Vite
- **Styling**: TailwindCSS
- **Icons**: Lucide React

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

## Features

- **GAI Interactive Platform**: Chat interface, document analysis, PPT generation, etc.
- **Smart Official Document Assistant**: Drafting tools for specific government use cases (e.g., mail responses, interior decoration review).
- **System Switching**: Toggle between GAI Platform and Smart Doc Assistant.
